Enterprise Fleet Intelligence V5.1

— An RNN-powered web application for predicting vehicle fuel efficiency and analyzing fleet cost

Key Features
-Single Vehicle Profiling: Predict MPG based on engine specs.

-Bulk Analytics: Process entire CSV/Excel fleet datasets.

-Executive Reporting: Generate downloadable PDF summaries.

-Interactive Visualizations: View efficiency trends via Plotly charts.

Tech Stack
1.Language: Python 3.11.

2.AI Framework: TensorFlow (Recurrent Neural Network).

3.Web Framework: Streamlit.

4.Data Handling: Pandas, NumPy, Scikit-Learn.
